Table Key:

Chaos Orb Value - Weighted value in Chaos orbs

7 Day Average - Average price from listings sold in last 7 days

Max Price - The highest observed listing yesterday

Min Price - The lowest observed listing yesterday

Exalted Orb Value - Converted value from Chaos Orb

Divine Orb Value - Converted Value from Chaos Orb


Endpoints used:

/status

https://api.poe.watch/status - Checks if the API is online, no parameters needed. Used to display the green/red API status message on the index page.


/hot

https://api.poe.watch/hot?league=Standard - Accepts a parameter, League, and outputs data similiar to /get with items with large market movement regardless of category. It is used by the trending table


/get

https://api.poe.watch/get?league=Standard&category=currency - Accepts 2 parameters, League and category, which then returns item details of a category such as, the name, icon, chaos, exalted, divine and history. This is used on the homepage to automatically display currency values and is manipulated by the user on the item search page to narrow down and search for specific items.
