About Me

API: api.poe.watch

Endpoints used:

https://api.poe.watch/status - Checks if the API is online, no parameters needed. Used to display the green/red API status message.

https://api.poe.watch/get?league=Standard&category=currency - Accepts 2 parameters, League and category, which then returns item details of a category such as, the name, icon, chaos, exalted, divine and history.

https://api.poe.watch/hot?league=Standard - Accepts a parameter, League, and outputs data similiar to /get with items with large market movement of any category.
